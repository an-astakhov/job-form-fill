# Anton Profile Package

This package mirrors the structure in `docs/example-profile-package` and is filled from repository sources including:

- CV templates and tailored CV variants
- base profile notes
- knowledge files
- past applications
- application-form notes
- past cover letters

## Goal

Produce two kinds of profile data:

1. Structured data for direct autofill
2. Rich Markdown notes for broader inference and future retrieval

## Files

- `profile.json`
- `experience-notes.md`
- `application-answer-bank.md`
- `role-targeting-notes.md`
- `fill-package-instructions.md`

## Source Notes

- Generated on 2026-06-16 from repo materials only
- Sensitive or uncertain answers such as salary expectations and notice period are intentionally left blank in JSON and marked for manual confirmation in Markdown
- Work-authorization wording reflects explicit repo evidence: EU citizenship and no visa sponsorship required
